# Requirements: CalendarPlanner v2.1

## Milestone

**v2.1 — Privacy, Reminders & Multi-Year Budget**

## Categories

- **PRIV**: Event Privacy (4 requirements)
- **REM**: Reminders (3 requirements)
- **BUD**: Multi-Year Budget (4 requirements)

## v1 Requirements (v2.1 Scope)

### Event Privacy

- **PRIV-01**: User can toggle event visibility between shared and private in the event creation/edit form
- **PRIV-02**: Private events are hidden from partner across all calendar view routes (day view, month grid, API)
- **PRIV-03**: Private events display a lock icon to the owner on the calendar grid
- **PRIV-04**: Changing event visibility from shared to private retracts it from partner's Google Calendar

### Reminders

- **REM-01**: User can enable/disable reminders with editable household defaults (30 min + 2 days before)
- **REM-02**: User can add and remove custom reminder entries (up to 5 per event) via chip-based UI
- **REM-03**: User can choose reminder method (popup or email) per reminder entry

### Multi-Year Budget

- **BUD-01**: User can navigate to any past or future year's budget dashboard using year arrows
- **BUD-02**: Each year's starting balance is computed from the prior year's ending balance (carry-forward)
- **BUD-03**: Recurring expenses only appear in years where they were active (year-scoped filtering)
- **BUD-04**: User can view year-over-year comparison summary with side-by-side annual totals and delta indicators
