Create `.github/workflows/e2e.yml` with:

```yaml
name: E2E Tests

on:
  pull_request:
    branches: [main]

jobs:
  e2e:
    timeout-minutes: 10
    runs-on: ubuntu-latest
    # Non-blocking: tests report results but don't block merge
    continue-on-error: true
    steps:
      - uses: actions/checkout@v4

      - uses: actions/setup-node@v4
        with:
          node-version: 'lts/*'
          cache: 'npm'

      - name: Install dependencies
        run: npm ci

      - name: Install Playwright Browsers
        run: npx playwright install chromium --with-deps

      - name: Run E2E tests
        run: npx playwright test --config=e2e/playwright.config.ts
        env:
          E2E_BASE_URL: ${{ secrets.E2E_BASE_URL }}
          E2E_FREE_EMAIL: ${{ secrets.E2E_FREE_EMAIL }}
          E2E_FREE_PASSWORD: ${{ secrets.E2E_FREE_PASSWORD }}
          E2E_PRO_EMAIL: ${{ secrets.E2E_PRO_EMAIL }}
          E2E_PRO_PASSWORD: ${{ secrets.E2E_PRO_PASSWORD }}
          E2E_FAMILY_PLUS_EMAIL: ${{ secrets.E2E_FAMILY_PLUS_EMAIL }}
          E2E_FAMILY_PLUS_PASSWORD: ${{ secrets.E2E_FAMILY_PLUS_PASSWORD }}

      - name: Upload test artifacts
        uses: actions/upload-artifact@v4
        if: ${{ !cancelled() }}
        with:
          name: e2e-results
          path: |
            e2e/test-results/
            e2e/playwright-report/
          retention-days: 14
```

Key decisions:
- Trigger: `pull_request` to `main` only (per CONTEXT.md decision)
- `continue-on-error: true` — non-blocking until suite stabilizes
- `timeout-minutes: 10` — generous for network-bound tests against Railway
- Install only `chromium` (not all browsers) to save CI time
- `--with-deps` installs OS-level dependencies needed for headless Chromium
- Artifacts: both test-results (screenshots/traces) and playwright-report (HTML)
- `retention-days: 14` — keep artifacts for 2 weeks
- Uses `actions/setup-node@v4` with npm cache for faster installs
- All 7 E2E secrets injected from GitHub Actions secrets
